
## instructions to build older versions using Ant



### master
```
$ mvn clean install
```
plug-in: `jcel-plugin/target/de.tudresden.inf.lat.jcel.jar`


### v0.22.0
```
$ mvn clean install
```
or
```
$ ant
```
plug-in: `jcel-plugin/target/de.tudresden.inf.lat.jcel.jar`


### v0.21.0
```
$ cd jcel/jcel-build
$ ant
```
plug-in: `jcel/jcel-build/jcel-plugin/target/de.tudresden.inf.lat.jcel.jar`
